# 🚀 Mini ERP CRM Management System

A full-stack ERP CRM application for managing customers, products, inventory, stock movement, and challans.

## Features

### Authentication
- JWT based login system
- Secure protected routes

### Customer Management
- Add customers
- View customers
- Update customer details

### Product Management
- Add products
- Track inventory
- Manage warehouse details

### Stock Management
- Stock IN/OUT tracking
- Automatic inventory updates

### Challan Management
- Generate challans
- Update stock automatically after confirmation

### Dashboard
- Customer statistics
- Product statistics
- Stock overview
- Sales tracking


## Tech Stack

Frontend:
- React.js
- TypeScript
- Axios
- CSS


Backend:
- Node.js
- Express.js
- TypeScript
- JWT Authentication


Database:
- MySQL


## Installation


Clone repository:

git clone YOUR_GITHUB_URL


Backend:

cd backend

npm install

npm run dev


Frontend:

cd frontend

npm install

npm run dev
